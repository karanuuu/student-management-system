# Student Assignment Manager

Overview
The Student Assignment Manager is a web-based application designed to help students and teachers organize tasks, track deadlines, and collaborate through inline comments.  
It features a clean, school-themed interface with a chalkboard background and colorful supplies, making the experience both functional and visually engaging.

 Features
Task Creation  
  Add new tasks with a title, description, priority level, and deadline.

  Task Listing 
  View all tasks in a structured list, styled with semi-transparent cards for readability.

  Inline Comments
  Each task includes a comment section where users can add and view feedback directly.

  Show/Hide Comments Toggle 
  Keep the task list compact by expanding or collapsing discussions.

  Delete Task (Trash Bin Icon)
  Remove tasks easily with a trash bin icon, complete with confirmation prompts.
